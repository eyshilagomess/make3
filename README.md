# Sistema de Estoque e Caixa - Make 3

Sistema desenvolvido para auxiliar na organização operacional da Make 3, com foco em controle de estoque, registro de vendas e acompanhamento do caixa.

## Objetivo

Criar uma solução simples e funcional para melhorar o controle interno do negócio, reduzindo processos manuais e facilitando o acompanhamento das movimentações.

## Funcionalidades

- Cadastro de produtos
- Controle de entrada e saída de estoque
- Registro de vendas
- Controle de caixa
- Acompanhamento de movimentações
- Organização dos dados para tomada de decisão

## Tecnologias e ferramentas utilizadas

- Lovable
- Inteligência Artificial generativa
- Desenvolvimento assistido por IA
- Estruturação de fluxos operacionais
- Interface web

## Meu papel no projeto

Fui responsável pela idealização, estruturação dos fluxos, organização das funcionalidades e desenvolvimento da solução com apoio de ferramentas de IA.
